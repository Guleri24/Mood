## Usage
### Enable mood

    python mood.py enable <mood_name>

### Add new mood:
    
    python mood.py add
Automatically open nano text editor.

Now you can add new moods in the `Add your moods here` section.

### Delete mood:

    python mood.py delete
### Help:

    python mood.py -h
